
# Encenação
## Planejamento 
Após a reunião geral do dia 15 de novembro, foi realizada uma reunião para a encenação. Nessa reunião o próposito da encenação foi improvisado na hora, já que teve realização do perfil de usuário quando desenvolvendo as personas.
Na encenação ultilizamos Otávio, a persona estudante para realizar somente duas tarefas simples:

1. Busca de turma.
2. Cadastrasr em um turma.
3. Vizualização de tarefas.
4. Realização/envio de tarefa



## Requisitos elicitados
